# cyf-cloud.front

## 安装依赖
```
npm install
```

或
```
cnpm install
```

## 启动
```
npm run serve_xxx
```
你可以针对你的环境编写一个配置文件：**.env.xxx**

## Build
```
npm run build    # Local
npm run build_gp # Github Page
```

### VUE_APP_BACKEND_SERVER_ADDR
后端的服务器地址，注意配置CORS与Cookie放行。

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
